# LogiFlow Warehouse Service

A microservice written in TypeScript/Express for managing LogiFlow warehouse storage nodes.

## Getting Started

### Prerequisites
- Node.js >= 18
- npm >= 9

### Installation
```bash
npm install
```

### Running Locally
To start the server in development mode (with auto-reload):
```bash
npm run dev
```

### Running Tests
To run integration and unit tests:
```bash
npm test
```
